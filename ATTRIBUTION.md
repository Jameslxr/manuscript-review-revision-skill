# 设计来源与归因

## Nature Skills

本项目在设计阶段参考了 [Yuan1z0825/nature-skills](https://github.com/Yuan1z0825/nature-skills) 的以下通用模式：

- 将 `SKILL.md` 作为执行入口，并把复杂规则拆分到按需读取的参考文件；
- 优先依据期刊官网、正式指南和一手学术来源，而不是泛化的写作偏好；
- 将可直接使用的科研产物和失败条件写入输出契约；
- 对缺失信息和无法核实的事项保持保守，不用猜测补齐。

Nature Skills 以 Apache License 2.0 发布。本仓库不包含其横幅、图片、二维码、演示资产或整份 Skill 文件；本项目的期刊感知路由、5–6 Agent 独立审稿、作者授权 Gate、Claim–Evidence–Reference 审计和投稿 Release Gate 为本仓库的独立实现。

## 独立性声明

- 本项目不是 Nature Portfolio 或 Springer Nature 的官方产品；
- 本项目与 Nature Skills 的维护者不存在官方隶属、背书或合作关系；
- “Nature”只在说明参考项目、期刊范围或用户明确调用相应能力时使用；
- 每个目标期刊的当前官方作者指南和编辑说明始终高于通用规则。
